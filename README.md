ALX Pre-course 

Description
I am now an ALX Student! This is my very first repository as a full-stack engineer. This project serves as an introduction to the fundamental tools I will be using throughout the Software Engineering program.

Focus Areas
In this pre-course, I am mastering:

* Git & GitHub: Learning version control from the command line.
* Unix Shell: Navigating the file system and managing files.
* Emacs/Vi: Getting comfortable with terminal based text editors.
* Professional Technologies: Setting up my local development environment. 

## Repository Structure
This repository contains the following initial projects:

* 0x00-shell_navigation: Basics of the Unix shell.
* 0x01-git: My first steps with Git, including creating branches and resolving conflicts. [10]
